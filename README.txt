This is the first udacity project for Machine Learning Nanodegree Program

There are three files in the repository:
(1) boston_housing_market_udacity.py
	This is the python code for the project that creates a Decision Tree Regression Model using Boston data provided in sklearn
(2) results.pdf
	This shows the histogram and performance of the Decision Tree Regression Model
(3) udacity_boston_housing_1.pdf
	This is report of the project providing answers to required questions and summarizing final outcome
